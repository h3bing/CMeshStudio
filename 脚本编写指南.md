# CMeshStudio 脚本编写指南

本指南将详细介绍如何使用CMeshStudio的脚本系统生成3D几何模型，以立方体（Cube）为例，包括实体特性表、脚本代码编写、文件保存和加载等内容。

## 一、实体特性表

每个实体都有一个特性表，用于存储和管理实体的属性。特性表按组分类，默认包含以下组：

| 组名 | 说明 |
|------|------|
| 常规 | 基本属性，如名称、ID等 |
| 外观 | 材质、颜色等外观属性 |
| 位姿 | 位置、旋转、缩放等变换属性 |
| 参数 | 几何生成相关的参数 |

### 立方体实体的特性表示例

| 组名 | 属性名 | 类型 | 默认值 | 说明 |
|------|--------|------|--------|------|
| 参数 | size | float | 1.0 | 立方体的大小 |
| 位姿 | position | PVector3 | (0, 0, 0) | 立方体的位置 |
| 位姿 | rotation | PVector3 | (0, 0, 0) | 立方体的旋转角度 |
| 位姿 | scale | PVector3 | (1, 1, 1) | 立方体的缩放比例 |
| 外观 | color | PVector3 | (1, 0.5, 0.2) | 立方体的颜色 |

## 二、脚本代码编写

### 1. 基本结构

每个脚本都必须包含一个 `generate()` 函数，这是脚本的入口点。基本结构如下：

```c
void generate() {
  // 脚本代码
}
```

### 2. 立方体生成脚本

以下是一个完整的生成立方体的脚本示例：

```c
void generate() {
  // 获取大小属性，默认为1.0
  float size = cgeo_get_prop_float("size");
  if (size <= 0) size = 1.0f;
  
  // 清除之前的几何数据
  cgeo_clear_vertices();
  cgeo_clear_indices();
  
  // 定义立方体的8个顶点
  // 前底面
  cgeo_add_vertex(-size, -size, -size);  // 顶点0
  cgeo_add_vertex(size, -size, -size);    // 顶点1
  // 前顶面
  cgeo_add_vertex(size, size, -size);     // 顶点2
  cgeo_add_vertex(-size, size, -size);    // 顶点3
  // 后底面
  cgeo_add_vertex(-size, -size, size);    // 顶点4
  cgeo_add_vertex(size, -size, size);     // 顶点5
  // 后顶面
  cgeo_add_vertex(size, size, size);      // 顶点6
  cgeo_add_vertex(-size, size, size);     // 顶点7
  
  // 定义立方体的6个面，每个面由2个三角形组成
  // 前面
  cgeo_add_index(0); cgeo_add_index(1); cgeo_add_index(2);
  cgeo_add_index(0); cgeo_add_index(2); cgeo_add_index(3);
  
  // 后面
  cgeo_add_index(4); cgeo_add_index(6); cgeo_add_index(5);
  cgeo_add_index(4); cgeo_add_index(7); cgeo_add_index(6);
  
  // 左面
  cgeo_add_index(4); cgeo_add_index(5); cgeo_add_index(1);
  cgeo_add_index(4); cgeo_add_index(1); cgeo_add_index(0);
  
  // 右面
  cgeo_add_index(3); cgeo_add_index(2); cgeo_add_index(6);
  cgeo_add_index(3); cgeo_add_index(6); cgeo_add_index(7);
  
  // 顶面
  cgeo_add_index(7); cgeo_add_index(6); cgeo_add_index(2);
  cgeo_add_index(7); cgeo_add_index(2); cgeo_add_index(3);
  
  // 底面
  cgeo_add_index(4); cgeo_add_index(0); cgeo_add_index(1);
  cgeo_add_index(4); cgeo_add_index(1); cgeo_add_index(5);
}
```

### 3. 脚本API详解

#### 几何操作函数

| 函数名 | 参数 | 说明 |
|--------|------|------|
| cgeo_add_vertex | float x, float y, float z | 添加一个顶点 |
| cgeo_add_index | int index | 添加一个索引 |
| cgeo_clear_vertices | 无 | 清除所有顶点 |
| cgeo_clear_indices | 无 | 清除所有索引 |

#### 属性操作函数

| 函数名 | 参数 | 说明 |
|--------|------|------|
| cgeo_get_prop_float | const char* name | 获取浮点属性 |
| cgeo_set_prop_float | const char* name, float value | 设置浮点属性 |
| cgeo_get_prop_int | const char* name | 获取整数属性 |
| cgeo_set_prop_int | const char* name, int value | 设置整数属性 |
| cgeo_get_prop_bool | const char* name | 获取布尔属性 |
| cgeo_set_prop_bool | const char* name, bool value | 设置布尔属性 |

#### 矩阵操作函数

| 函数名 | 参数 | 说明 |
|--------|------|------|
| cgeo_matrix_identity | 无 | 设置单位矩阵 |
| cgeo_matrix_translate | float x, float y, float z | 平移矩阵 |
| cgeo_matrix_rotate_x | float angle | 绕X轴旋转 |
| cgeo_matrix_rotate_y | float angle | 绕Y轴旋转 |
| cgeo_matrix_rotate_z | float angle | 绕Z轴旋转 |
| cgeo_matrix_scale | float x, float y, float z | 缩放矩阵 |

## 三、文件保存与加载

### 1. 保存实体

CMeshStudio使用JSON格式保存实体，文件后缀为 `.cme`。保存时会包含以下信息：

- 实体ID
- 实体名称
- 特性表（按组分类）
- 脚本代码
- 变换矩阵

### 2. 加载实体

当加载一个 `.cme` 文件时，系统会：

1. 解析JSON文件
2. 重建实体对象
3. 恢复特性表
4. 恢复脚本代码
5. 恢复变换矩阵
6. 自动执行脚本生成几何

## 四、脚本编写流程

### 1. 创建新实体

1. 点击左侧工具栏的"新实体"按钮
2. 在属性编辑器中设置实体名称和属性
3. 在脚本编辑器中编写几何生成脚本

### 2. 编写脚本

1. 定义需要的属性（如size）
2. 清除旧的几何数据
3. 添加顶点
4. 添加索引
5. 保存脚本

### 3. 运行脚本

1. 点击脚本编辑器工具栏的"运行"按钮
2. 查看视口中的几何效果
3. 调整属性值，重新运行脚本

### 4. 保存实体

1. 点击左侧工具栏的"保存"按钮
2. 选择保存位置，文件后缀为 `.cme`
3. 点击"保存"按钮

### 5. 加载实体

1. 点击左侧工具栏的"打开"按钮
2. 选择 `.cme` 文件
3. 点击"打开"按钮
4. 实体将自动加载并生成几何

## 五、高级技巧

### 1. 使用循环生成复杂几何

```c
void generate() {
  int segments = (int)cgeo_get_prop_float("segments");
  if (segments < 3) segments = 3;
  
  float radius = cgeo_get_prop_float("radius");
  if (radius <= 0) radius = 1.0f;
  
  cgeo_clear_vertices();
  cgeo_clear_indices();
  
  // 生成顶点
  for (int i = 0; i <= segments; i++) {
    float angle = 2.0f * 3.14159f * i / segments;
    float x = radius * cos(angle);
    float y = 0;
    float z = radius * sin(angle);
    cgeo_add_vertex(x, y, z);
  }
  
  // 生成索引
  for (int i = 0; i < segments; i++) {
    cgeo_add_index(0);     // 中心点
    cgeo_add_index(i + 1);  // 当前点
    cgeo_add_index(i + 2);  // 下一个点
  }
}
```

## 六、常见问题与解决方案

### 1. 脚本编译错误

**症状**：控制台显示编译错误
**解决方案**：检查脚本语法，确保所有括号和分号都正确

### 2. 几何体不显示

**症状**：脚本执行成功，但视口中没有显示几何体
**解决方案**：
- 检查是否添加了顶点和索引
- 检查顶点坐标是否在合理范围内
- 检查相机位置是否能看到几何体

### 3. 几何体显示异常

**症状**：几何体显示但形状不正确
**解决方案**：
- 检查顶点顺序
- 检查索引是否正确
- 检查渲染模式是否合适

### 4. 脚本执行错误

**症状**：控制台显示运行时错误
**解决方案**：
- 检查是否使用了未定义的变量
- 检查数组索引是否越界
- 检查是否除以零
- 检查函数调用参数是否正确

### 5. 属性访问错误

**症状**：属性值未生效或显示默认值
**解决方案**：
- 检查属性名称是否正确
- 检查属性类型是否匹配
- 确保属性已在特性表中定义

### 2. 球体（Sphere）

```c
void generate() {
  int segments = (int)cgeo_get_prop_float("segments");
  if (segments < 3) segments = 3;
  
  float radius = cgeo_get_prop_float("radius");
  if (radius <= 0) radius = 1.0f;
  
  cgeo_clear_vertices();
  cgeo_clear_indices();
  
  // 生成顶点
  for (int i = 0; i <= segments; i++) {
    float theta = 3.14159f * i / segments;
    float sinTheta = sin(theta);
    float cosTheta = cos(theta);
    
    for (int j = 0; j <= segments; j++) {
      float phi = 2.0f * 3.14159f * j / segments;
      float sinPhi = sin(phi);
      float cosPhi = cos(phi);
      
      float x = radius * sinTheta * cosPhi;
      float y = radius * cosTheta;
      float z = radius * sinTheta * sinPhi;
      
      cgeo_add_vertex(x, y, z);
    }
  }
  
  // 生成索引
  for (int i = 0; i < segments; i++) {
    for (int j = 0; j < segments; j++) {
      int first = i * (segments + 1) + j;
      int second = first + segments + 1;
      
      cgeo_add_index(first);
      cgeo_add_index(second);
      cgeo_add_index(first + 1);
      
      cgeo_add_index(second);
      cgeo_add_index(second + 1);
      cgeo_add_index(first + 1);
    }
  }
}
```
