# CMeshStudio 脚本编写指南

本指南将详细介绍如何使用CMeshStudio的脚本系统生成3D几何模型，包括实体特性表、脚本代码编写、文件保存和加载等内容。

## 一、实体特性表

每个实体都有一个特性表，用于存储和管理实体的属性。特性表按组分类，默认包含以下组：

| 组名 | 说明 |
|------|------|
| 常规 | 基本属性，如名称、ID等 |
| 外观 | 材质、颜色等外观属性 |
| 位姿 | 位置、旋转、缩放等变换属性 |
| 参数 | 几何生成相关的参数 |

## 二、脚本代码编写

### 1. 基本结构

每个脚本都必须包含一个 `generate()` 函数，这是脚本的入口点。基本结构如下：

```c
void generate() {
  // 脚本代码
}
```

### 2. 脚本API详解

#### 几何操作函数

| 函数名 | 参数 | 说明 |
|--------|------|------|
| cgeo_add_vertex | float x, float y, float z | 添加一个顶点 |
| cgeo_set_normal | float x, float y, float z | 设置当前顶点的法线向量 |
| cgeo_add_index | int index | 添加一个索引 |
| cgeo_clear_vertices | 无 | 清除所有顶点 |
| cgeo_clear_indices | 无 | 清除所有索引 |

#### 属性操作函数

| 函数名 | 参数 | 说明 |
|--------|------|------|
| cgeo_get_prop_float | const char* name | 获取浮点属性 |
| cgeo_set_prop_float | const char* name, float value | 设置浮点属性 |
| cgeo_get_prop_int | const char* name | 获取整数属性 |
| cgeo_set_prop_int | const char* name, int value | 设置整数属性 |
| cgeo_get_prop_bool | const char* name | 获取布尔属性 |
| cgeo_set_prop_bool | const char* name, bool value | 设置布尔属性 |

#### 数学函数

| 函数名 | 参数 | 说明 |
|--------|------|------|
| cgeo_sin | float x | 正弦函数 |
| cgeo_cos | float x | 余弦函数 |
| cgeo_sqrt | float x | 平方根函数 |

#### 矩阵操作函数

| 函数名 | 参数 | 说明 |
|--------|------|------|
| cgeo_matrix_identity | 无 | 设置单位矩阵 |
| cgeo_matrix_translate | float x, float y, float z | 平移矩阵 |
| cgeo_matrix_rotate_x | float angle | 绕X轴旋转 |
| cgeo_matrix_rotate_y | float angle | 绕Y轴旋转 |
| cgeo_matrix_rotate_z | float angle | 绕Z轴旋转 |
| cgeo_matrix_scale | float x, float y, float z | 缩放矩阵 |

## 三、文件保存与加载

### 1. 保存实体

CMeshStudio使用JSON格式保存实体，文件后缀为 `.cme`。保存时会包含以下信息：

- 实体ID
- 实体名称
- 特性表（按组分类）
- 脚本代码
- 变换矩阵

### 2. 加载实体

当加载一个 `.cme` 文件时，系统会：

1. 解析JSON文件
2. 重建实体对象
3. 恢复特性表
4. 恢复脚本代码
5. 恢复变换矩阵
6. 自动执行脚本生成几何

### 3. 示例.cme文件结构

### 4. JSON格式注意事项

- 脚本代码中的换行符必须使用 `\n` 转义，不能使用实际的换行符
- 字符串中的引号必须使用 `\"` 转义
- 确保JSON格式正确，否则会导致加载失败

## 四、脚本编写流程

### 1. 创建新实体

1. 点击左侧工具栏的"新实体"按钮
2. 在属性编辑器中设置实体名称和属性
3. 在脚本编辑器中编写几何生成脚本

### 2. 编写脚本

1. 定义需要的属性（如size、radius等）
2. 清除旧的几何数据
3. 添加顶点并设置法线
4. 添加索引
5. 保存脚本

### 3. 运行脚本

1. 点击脚本编辑器工具栏的"运行"按钮
2. 查看视口中的几何效果
3. 调整属性值，重新运行脚本

### 4. 保存实体

1. 点击左侧工具栏的"保存"按钮
2. 选择保存位置，文件后缀为 `.cme`
3. 点击"保存"按钮

### 5. 加载实体

1. 点击左侧工具栏的"打开"按钮
2. 选择 `.cme` 文件
3. 点击"打开"按钮
4. 实体将自动加载并生成几何

## 五、高级技巧

### 1. 法线计算

法线向量对于正确的光照效果非常重要。以下是一些常见几何体的法线计算方法：

- **球体**：法线向量就是顶点位置的单位向量
- **圆柱体**：侧面法线指向径向外侧，顶部/底部法线垂直于表面
- **圆锥体**：侧面法线指向外侧，底部法线垂直于表面

### 2. 使用循环生成复杂几何

```c
void generate() {
  int segments = cgeo_get_prop_int("分段数");
  if (segments < 3) segments = 3;
  
  float radius = cgeo_get_prop_float("半径");
  if (radius <= 0) radius = 1.0f;
  
  cgeo_clear_vertices();
  cgeo_clear_indices();
  
  // 生成顶点
  for (int i = 0; i <= segments; i++) {
    float angle = 2.0f * 3.14159f * i / segments;
    float x = radius * cgeo_cos(angle);
    float y = 0;
    float z = radius * cgeo_sin(angle);
    cgeo_add_vertex(x, y, z);
    cgeo_set_normal(x / radius, 0.0f, z / radius); // 法线指向外侧
  }
  
  // 生成索引
  for (int i = 0; i < segments; i++) {
    cgeo_add_index(0);     // 中心点
    cgeo_add_index(i + 1);  // 当前点
    cgeo_add_index(i + 2);  // 下一个点
  }
}
```

## 六、常见问题与解决方案

### 1. 脚本编译错误

**症状**：控制台显示编译错误
**解决方案**：检查脚本语法，确保所有括号和分号都正确

### 2. 几何体不显示

**症状**：脚本执行成功，但视口中没有显示几何体
**解决方案**：
- 检查是否添加了顶点和索引
- 检查顶点坐标是否在合理范围内
- 检查相机位置是否能看到几何体

### 3. 几何体显示异常

**症状**：几何体显示但形状不正确
**解决方案**：
- 检查顶点顺序
- 检查索引是否正确
- 检查渲染模式是否合适

### 4. 脚本执行错误

**症状**：控制台显示运行时错误
**解决方案**：
- 检查是否使用了未定义的变量
- 检查数组索引是否越界
- 检查是否除以零
- 检查函数调用参数是否正确

### 5. 属性访问错误

**症状**：属性值未生效或显示默认值
**解决方案**：
- 检查属性名称是否正确
- 检查属性类型是否匹配
- 确保属性已在特性表中定义

### 6. JSON格式错误

**症状**：加载实体时属性和脚本为空
**解决方案**：
- 确保脚本代码中的换行符使用 `\n` 转义
- 确保字符串中的引号使用 `\"` 转义
- 验证JSON格式是否正确

### 2. 球体（Sphere）

```c
void generate() {
  float radius = cgeo_get_prop_float("半径");
  if (radius <= 0) radius = 1.0f;
  
  int segments = cgeo_get_prop_int("分段数");
  if (segments < 3) segments = 3;
  
  cgeo_clear_vertices();
  cgeo_clear_indices();
  
  // 生成顶点
  for (int i = 0; i <= segments; i++) {
    float theta = 3.14159f * i / segments;
    float sinTheta = cgeo_sin(theta);
    float cosTheta = cgeo_cos(theta);
    
    for (int j = 0; j <= segments; j++) {
      float phi = 2.0f * 3.14159f * j / segments;
      float sinPhi = cgeo_sin(phi);
      float cosPhi = cgeo_cos(phi);
      
      float x = radius * sinTheta * cosPhi;
      float y = radius * cosTheta;
      float z = radius * sinTheta * sinPhi;
      
      cgeo_add_vertex(x, y, z);
      
      // 计算法向量（球体的法向量就是归一化的位置向量）
      float nx = sinTheta * cosPhi;
      float ny = cosTheta;
      float nz = sinTheta * sinPhi;
      cgeo_set_normal(nx, ny, nz);
    }
  }
  
  // 生成索引
  for (int i = 0; i < segments; i++) {
    for (int j = 0; j < segments; j++) {
      int first = i * (segments + 1) + j;
      int second = first + segments + 1;
      
      cgeo_add_index(first);
      cgeo_add_index(second);
      cgeo_add_index(first + 1);
      
      cgeo_add_index(second);
      cgeo_add_index(second + 1);
      cgeo_add_index(first + 1);
    }
  }
}
```

### 3. 圆柱体（Cylinder）

```c
void generate() {
  float radius = cgeo_get_prop_float("半径");
  if (radius <= 0) radius = 1.0f;
  
  float height = cgeo_get_prop_float("高度");
  if (height <= 0) height = 2.0f;
  
  int segments = cgeo_get_prop_int("分段数");
  if (segments < 3) segments = 3;
  
  cgeo_clear_vertices();
  cgeo_clear_indices();
  
  float halfHeight = height / 2.0f;
  
  // 生成顶部和底部顶点
  for (int i = 0; i <= segments; i++) {
    float angle = 2.0f * 3.14159f * i / segments;
    float x = radius * cgeo_cos(angle);
    float z = radius * cgeo_sin(angle);
    
    // 计算侧面法线（指向外）
    float normalX = x / radius;
    float normalZ = z / radius;
    
    // 顶部顶点
    cgeo_add_vertex(x, halfHeight, z);
    cgeo_set_normal(0.0f, 1.0f, 0.0f); // 顶部顶点法向量
    
    // 底部顶点
    cgeo_add_vertex(x, -halfHeight, z);
    cgeo_set_normal(0.0f, -1.0f, 0.0f); // 底部顶点法向量
    
    // 侧面顶部顶点（使用侧面法线）
    cgeo_add_vertex(x, halfHeight, z);
    cgeo_set_normal(normalX, 0.0f, normalZ); // 侧面法线
    
    // 侧面底部顶点（使用侧面法线）
    cgeo_add_vertex(x, -halfHeight, z);
    cgeo_set_normal(normalX, 0.0f, normalZ); // 侧面法线
  }
  
  // 生成顶部和底部面
  for (int i = 0; i < segments; i++) {
    // 顶部面
    cgeo_add_index(0);
    cgeo_add_index(4 * i + 2);
    cgeo_add_index(4 * ((i + 1) % segments) + 2);
    
    // 底部面
    cgeo_add_index(1);
    cgeo_add_index(4 * i + 3);
    cgeo_add_index(4 * ((i + 1) % segments) + 3);
  }
  
  // 生成侧面
  for (int i = 0; i < segments; i++) {
    int top1 = 4 * i + 4;
    int top2 = 4 * ((i + 1) % segments) + 4;
    int bottom1 = 4 * i + 5;
    int bottom2 = 4 * ((i + 1) % segments) + 5;
    
    cgeo_add_index(top1);
    cgeo_add_index(bottom1);
    cgeo_add_index(top2);
    
    cgeo_add_index(top2);
    cgeo_add_index(bottom1);
    cgeo_add_index(bottom2);
  }
}
```

### 4. 圆锥体（Cone）

```c
void generate() {
  float radius = cgeo_get_prop_float("半径");
  if (radius <= 0) radius = 1.0f;
  
  float height = cgeo_get_prop_float("高度");
  if (height <= 0) height = 2.0f;
  
  int segments = cgeo_get_prop_int("分段数");
  if (segments < 3) segments = 3;
  
  cgeo_clear_vertices();
  cgeo_clear_indices();
  
  float halfHeight = height / 2.0f;
  
  // 生成顶点
  // 顶部顶点
  cgeo_add_vertex(0.0f, halfHeight, 0.0f);
  cgeo_set_normal(0.0f, 1.0f, 0.0f); // 顶部顶点法向量
  
  // 底部顶点
  for (int i = 0; i <= segments; i++) {
    float angle = 2.0f * 3.14159f * i / segments;
    float x = radius * cgeo_cos(angle);
    float z = radius * cgeo_sin(angle);
    cgeo_add_vertex(x, -halfHeight, z);
    // 为底部顶点设置侧面法线（指向外）
    float normalX = x / radius;
    float normalZ = z / radius;
    cgeo_set_normal(normalX, 0.0f, normalZ); // 侧面法线
  }
  
  // 生成侧面
  for (int i = 0; i < segments; i++) {
    cgeo_add_index(0);
    cgeo_add_index(i + 1);
    cgeo_add_index(i + 2);
  }
  
  // 生成底部面
  // 重新添加底部顶点，使用向下的法线
  int bottomCenter = segments + 2;
  cgeo_add_vertex(0.0f, -halfHeight, 0.0f);
  cgeo_set_normal(0.0f, -1.0f, 0.0f); // 底部中心法线
  
  for (int i = 1; i <= segments; i++) {
    cgeo_add_index(bottomCenter);
    cgeo_add_index(i);
    cgeo_add_index(i + 1);
  }
}
```
